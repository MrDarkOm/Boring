# Вайбматч 🎲

Прототип мобильного приложения для подбора занятия «прямо сейчас» —
свайп-карточки в стиле Tinder для фильмов, мест, скидок, книг и игр.

## Что внутри

- **Свайп-карточки** — влево «нет», вправо «да», вверх «на потом», + кнопка undo
- **Геолокация и реальная погода** — Open-Meteo API (без ключа), погода влияет на подборку
- **Карта рядом** — реальные расстояния по координатам (Haversine)
- **AI-подсказки** — каждые 3 свайпа Claude советует что попробовать (через прокси)
- **Совместный режим** — два человека свайпают, находится пересечение
- **Статистика, сохранённые с заметками, настройки, «Удиви меня»**

## Стек

Vite + React + TypeScript. Без тяжёлых зависимостей — стили инлайновые,
анимации на CSS.

## Запуск

```bash
npm install
```

### Вариант 1 — без AI (быстро посмотреть UI)

```bash
npm run dev
```

Откроется на http://localhost:5173. Всё работает, кроме AI-подсказок
(запрос на /api/ai-tip вернёт ошибку — она проглатывается, приложение не падает).

### Вариант 2 — с AI-подсказками

AI-подсказки идут через локальный прокси-сервер, чтобы ключ Anthropic
**не попадал в браузер**.

1. Скопируй `.env.example` в `.env` и вставь свой ключ:

   ```
   ANTHROPIC_API_KEY=sk-ant-...
   ```

2. Запусти прокси и фронтенд вместе:

   ```bash
   npm run dev:all
   ```

   Либо в двух терминалах: `npm run server` и `npm run dev`.

Прокси слушает порт 8787, Vite проксирует на него запросы `/api/*`
(см. `vite.config.ts`).

> ⚠️ Никогда не клади ключ в код фронтенда или в переменные `VITE_*` —
> они попадают в бандл и видны всем. Ключ читается только сервером.

## Структура

```
src/
  data/        — карточки, настроения, места, погода
  types/       — TypeScript-типы
  lib/         — хелперы (расстояния, WMO-коды, вызов AI)
  hooks/       — useGeoWeather (геолокация + погода)
  components/  — переиспользуемые: NavBar, Toast, ui (Glow, Tag)
  screens/     — экраны: Splash, Onboarding, Swipe, Match,
                 Saved, Stats, Map, Settings, Coop, SurpriseModal
  App.tsx      — оркестратор состояния и роутинг между экранами
server/
  proxy.mjs    — прокси для Anthropic API (без зависимостей, Node 18+)
```

## Работа в Claude Code

Открой эту папку и запусти `claude`. Дальше можно давать задачи обычным
текстом — например «добавь время суток как фактор подбора» или
«подключи TMDB для фильмов». Claude Code сам отредактирует нужные файлы.
