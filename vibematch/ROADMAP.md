# 🚀 Вайбматч — дорожная карта до продакшена

**Стек решения:** Capacitor (iOS/Android из веб-кода) · Supabase (Postgres + Auth + Realtime + Storage) · TMDB (фильмы) · Overpass/OSM (места)

Цель: превратить UI-прототип в публикуемое в сторах приложение с реальными
данными, сохранением состояния, аккаунтами и рабочим совместным режимом.

## Новые механики (бэклог фич)

- **«Решено за тебя»** — один AI-ответ без свайпов (есть в бэклоге)
- **Стрики и достижения** — геймификация ретеншена
- **Коллекции/списки** — группировать saved («на выходные», «свидание»)
- **Соц-граф** — друзья, общие мэтчи, рекомендации друзей
- **Время суток и календарь** как факторы подбора
- **Бронирование/покупка** прямо из карточки (партнёрки, скидки)
- **Push «скидка рядом истекает»** на базе гео + времени
